# movie-finder-app

Movie Finder Android App named "Seyret"

FEATURES
-Sign In/Sign Up
-Authentication
-Integrated with Firebase Realtime Database
-Searching specific movies
-Rating movies
-Adding a movie to favourites
-Movies listed by category
-Customizable user account
